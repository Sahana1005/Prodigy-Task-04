Projects

MySos

MySos: Personal Emergency SOS App Description: MySos is an innovative emergency SOS app that allows users to send distress signals and their location to predefined contacts in case of an emergency. Developed using MIT App Inventor, this app leverages simple drag-and-drop programming to create an accessible, user-friendly interface. The core functionality of the app is to provide peace of mind by allowing users to send an emergency alert with just a tap, making it easier to get help in urgent situations.
Technologies Used:
MIT App Inventor: A visual programming platform that allowed for easy and quick app development with a drag-and-drop interface.
Geolocation API: The app uses the device’s built-in GPS to track the user's real-time location.
SMS API: Sends automated SMS to predefined contacts with the user's location and a distress message.
Features:
SOS Button: The app includes a large, easy-to-press SOS button that sends an alert to emergency contacts.
Real-time Location Sharing: Once the SOS button is activated, the app automatically grabs the user’s GPS coordinates and sends them with the message.
Customizable Contacts: Users can easily add emergency contacts who will receive the SOS message when triggered.
Emergency Message: Predefined emergency messages can be sent alongside the user's location, including customizable text to provide more context if needed.
